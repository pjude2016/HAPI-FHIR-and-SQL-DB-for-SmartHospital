This is a copy of the UCL Smart Hospital project database.
It contains all working tables and data up to 18-03-2019.

The .bacpac file must be updated to Azure Storage as blob data.
Then an Azure Data Storage service must be created to later on import a database from the .bacpac file stored in Azure Storage.

Fur further details on setting up please follow the steps in the following guide:
https://github.com/Huachao/azure-content/blob/master/articles/sql-database/sql-database-import.md
